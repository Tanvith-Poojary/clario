Clario
Clario is a calm, AI-powered clarity and emotional support platform designed to help people understand their thoughts, navigate life challenges, and gain direction—without noise, pressure, or toxic positivity.
 Features
 AI Motivator (Listen-First)
Reflective conversations that understand before guiding
No generic advice or forced positivity
 Problem-Based Journeys
Real-life situations like career confusion, feeling lost, or emotional pressure
Structured guidance → clarity → action
 Clarity Score
Tracks self-understanding and decision confidence
Focus on clarity, not happiness
 Daily Micro-Actions
One small, meaningful action per day
No streaks, no pressure
 Anonymous Community
Safe, calm space
No likes, no followers, no social pressure
 Unsent Letters
Private emotional release
Write without judgment
 Philosophy
“Clario listens before it guides.”
Clario is not a therapy app or a social media platform.
It is built to create a safe, quiet, and supportive environment where users can gain clarity and emotional balance.
 Tech Stack
Frontend: React / Vite
Styling: Tailwind CSS
AI Integration: OpenAI API
Backend: Firebase
